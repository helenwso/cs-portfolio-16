Motherboard

What is a motherboard?

A motherboard is a circuit board in a computer that allots power, holds the most crucial components to the computer's processing (including the CPU, RAM, etc) and allows those components to communicate with one another. Computers of the 1980s allowed for replacement motherboards, which let computers receive upgrades. In later years, more and more crucial components were moved to the motherboard, and in the next decade, motherboards were created that could not be upgraded. Because of their high energy use, they generate a lot of heat, which when not properly dealt with can cause crashing, damage, or other problems. Therefore, most computers nowadays have heat sinks and CPU fans.

























CPU

What is a CPU?

I've mentioned CPU a couple times, but what does it really mean? Well, CPU stands for Central Processing Unit, or more simply put, processor. The CPU is the computer's circuitry, following directions to calculate the operations of a computer program. Because the CPU performs important functions, it is mass produced to appear in most digital devices in a versatile yet complex form.





















RAM



What is RAM?

RAM stands for Random Access Memory. In computer lingo, memory is a device that stores temporary memory. That means that when your computer is turned off, this information also disappears. The RAM is used to store information temporarily, and "randomly access" it, so that the computer can run faster during each session, in contrast to hard drives for example, where speed of writing and reading data is changed depending on the physical location of the data.






















Communication

How do different components communicate?

The different components of a computer can communicate because they are plugged into ports and slots in the motherboard, which powers and connects everything. These components are also physically connected by something called a bus, as it contains things like cables and circuits, which can transmit information. There are two general types of buses: one for the CPU and RAM, called the internal bus; and one for other motherboard components called the Expansion Bus.

























Keyboard, Mouse, and Monitor

What are the keyboard, mouse, and monitor?

The keyboard is the part of the computer which controls text input by the user, and is either attached, onscreen, or has to be plugged into the computer.

The mouse controls the movement of the cursor on a computer's screen, and often includes left and right click buttons and a wheel for scrolling in the middle. Cursor movement and button/scrolling all help the user give the computer commands to perform.

The monitor is the graphics display of the computer, usually referring to the screen. The monitor is measured in pixels. A monitor resolution expresses pixel density, and the more dense the pixels, the clearer the images is.
